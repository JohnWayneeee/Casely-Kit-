
# /casely.clear

## Назначение
Сброс к дефолтным настройкам. Удаляет персональные правила и при выборе очищает данные.

## Синтаксис
```
/casely.clear
```

## Алгоритм
1. Предупредить о последствиях и попросить выбрать вариант:
   - Только правила (`user-rules.md`)
   - Правила + examples
   - Правила + requirements
   - Правила + output
   - Полная очистка (rules + examples + requirements + output)
   - Отмена
2. Удалить выбранные элементы, сохранить пустые папки с `.gitkeep`.
3. Сообщить, что теперь используется fallback `.casely/config/default-rules.md`.

## Шаблон подтверждения
```
⚠️ Очистка:
- user-rules.md: удалить
- examples: {да/нет}
- requirements: {да/нет}
- output: {да/нет}
Подтвердите вариант или отмените.
```

## Шаблон успешного ответа
```
✅ Очистка завершена.
Удалено: {перечень}
Состояние: вернулись к default-rules.
Для нового стиля: добавьте примеры в .casely/examples/ и выполните /casely.init
```
